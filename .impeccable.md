# .impeccable.md — BTCナラティブ配信サイト デザインコンテキスト

## Design Context

### Users
日本語のリテール層（個人投資家・クリプト関心層）。スマホ／コーヒー片手で、6時間ごとに更新される「BTCの物語＝認知」の分析を読む。トレード画面ではなく、落ち着いて読む“読み物”。**可読性が最優先**。

### Brand Personality
3語：**明晰・誠実・専門的（clear / honest / professional）**。
ナラティブに乗るのではなく観察する立場。断定しない・煽らない。知的だが堅すぎない、信頼できるテックメディアのトーン。

### Aesthetic Direction（2026-06-07 改訂）
**参考＝[ics.media](https://ics.media/)**：明るくクリーンなテックメディア。**ライトテーマ・サンセリフ・余白広め・縦並びの読みやすい記事リスト・落ち着いた青系アクセント**。
- テーマ：**ライト**（near-white背景／濃いスレートの文字）。※初版のダーク明朝は「巨大で見にくい」ため撤回。
- 個性：装飾でなく**端正な可読性とクリアな階層**で差別化。
- アンチリファレンス：巨大明朝見出し、ダーク＋ネオン、紫グラデ、グラデ文字、同型カードの過密グリッド、左帯ボーダー、グラスモーフィズム。

### Typography（サンセリフ・反射定番を除外）
- 見出し：**Zen Kaku Gothic New（700/900）** — モダンで存在感ある日本語ゴシック。
- 本文：**Murecho（400/500/600）** — 温かく読みやすい humanist 日本語サンセリフ。
- 見出しは控えめなサイズに（巨大化させない）。本文70ch幅。数値はtabular-nums。

### Color（OKLCH・ライト・neutralはアクセント側へ微tint・60-30-10）
- 背景：`oklch(0.99 0.003 220)`（near-white・微クール）
- サーフェス：`oklch(0.975 0.004 220)` / `oklch(0.955 0.006 220)`
- 文字：`oklch(0.30 0.012 235)`（純黒は使わない）
- 副次：`oklch(0.50 0.012 235)` / 微弱：`oklch(0.62 0.010 235)`
- ヘアライン：`oklch(0.91 0.005 230)`
- アクセント（落ち着いた青〜スレートティール）：`oklch(0.55 0.085 215)`、リンクは contrast 確保で `oklch(0.50 0.09 220)`

### Design Principles
1. **可読性が最優先**：読みやすさを損なう装飾はしない。
2. **クリアな階層**：見出し/本文/メタを明快に。巨大化で圧をかけない。
3. **余白で落ち着き**：ゆったり、テックメディア的に端正。
4. **データは静かに正確に**：計器盤はtabular-nums・ヘアライン。
5. **アンチ・スロップ**：左帯・グラデ文字・ネオン・同型カード過密は踏まない。
