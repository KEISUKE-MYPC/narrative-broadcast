# .impeccable.md — Narrative Broadcast デザインコンテキスト

## Design Context

### Users
日本語のリテール層（個人投資家・クリプト/マーケット関心層）。スマホ／コーヒー片手で、6時間ごとに更新される「市場参加者の物語＝認知」の構造分析を読む。トレード画面ではなく、落ち着いて読む“読み物”。**可読性が最優先**。
※当面はBTCのみだが、将来は他分野（ETH／マクロ等）にも拡張する前提。

### Brand Personality
3語：**明晰・誠実・専門的（clear / honest / professional）**。
ナラティブに乗るのではなく観察する立場。断定しない・煽らない。知的だが堅すぎない、信頼できる観測メディアのトーン。

### Aesthetic Direction（2026-06-07 ダークへ刷新・確定）
**参考＝[ics.media の記事ページ](https://ics.media/entry/260603/)**：落ち着いたダーク基調の読み物系テックメディア。
- テーマ：**ダーク**（深いスレート背景／高明度の文字）。夜・集中して読む文脈に合わせる。
  ※変遷＝初版ダーク明朝(撤回)→ライト(撤回)→**現ダーク・サンセリフ**。
- 個性：**端正な可読性＋静かなデータ表現**で差別化。ヒーローは“ブロードキャスト・コンソール”＝微細な縦ヘアライングリッド＋ティールの奥行きグロー＋上端アクセントライン。
- **重要な線引き**：ダークだが「ダーク＋ネオン／グロー連発」のAIスロップにはしない。アクセントは**ティール1色を控えめに**（60-30-10の10）。彩度は明度の端で必ず下げる。
- アンチリファレンス：巨大明朝見出し、紫→青グラデ、グラデ文字、ネオン乱用、同型カードの過密グリッド、左/右帯ボーダー(>1px)、グラスモーフィズム、巨大メトリクスのテンプレ。

### Typography（サンセリフ・反射定番を除外）
- 見出し：**Zen Kaku Gothic New（700/900）** — モダンで存在感ある日本語ゴシック。
- 本文：**Murecho（400/500/600）** — 温かく読みやすい humanist 日本語サンセリフ。
- 見出しは巨大化させない。本文は読み幅を確保（measure 70ch）。数値は tabular-nums。
- ライト文字 on ダークは“軽く”見えるため行間は気持ち広め。フォントはGoogle Fonts直リンク（CJK確実描画）。

### Color（OKLCH・ダーク・neutralはアクセント側へ微tint・60-30-10）
実値は `app/globals.css` の `:root`。
- 背景：`oklch(0.17 0.013 250)`／サーフェス：`oklch(0.215 0.015 250)` / `oklch(0.255 0.016 250)`
- 文字：`oklch(0.93 0.008 240)`（純白は使わない）／副次：`oklch(0.72 0.011 240)`／微弱：`oklch(0.56 0.011 240)`
- ヘアライン：`oklch(0.31 0.013 250)`
- アクセント（ティール）：`oklch(0.78 0.10 205)`／リンク `oklch(0.82 0.10 205)`／wash `oklch(0.32 0.055 210)`
- ヒーロー/マストヘッド地：`oklch(0.205 0.022 240)`〜`oklch(0.155 0.026 245)`、上の文字 `oklch(0.965 0.006 230)`、グロー `oklch(0.72 0.10 205)`

### 分野(カテゴリ)の見せ方
将来の複数分野に備え、「今どの分野を見ているか」を静かに明示する。
- **分野チップ**（カード・記事・パンくず）＝`var(--accent-wash)` 地＋`--accent-ink` 文字の小バッジ。
- **パンくず**（記事上部）／**SiteNav**（分野2つ以上で自動表示・現在地ハイライト）。
- 分野ごとの識別色はレジストリ（`lib/categories.ts`）の `accent` で管理。色は派手にせず、ティール基調から逸脱しすぎない範囲で。

### Design Principles
1. **可読性が最優先**：読みやすさを損なう装飾はしない。
2. **クリアな階層**：見出し/本文/メタ/分野を明快に。巨大化や強い色で圧をかけない。
3. **静かなデータ表現**：計器盤・チャートは tabular-nums・ヘアライン中心。
4. **アクセントは希少に**：ティール1色を要所だけ。グロー/ネオンに逃げない。
5. **アンチ・スロップ**：帯ボーダー・グラデ文字・ネオン乱用・同型カード過密・巨大メトリクステンプレは踏まない。
6. **拡張に開く**：分野が増えても破綻しない構造（チップ／ナビ／レジストリ駆動）を保つ。
